### Content Strategy — Udemy 50% Rule মেনে

**একই থাকবে (Udemy গণনা করে না):**
- Slide layout ও design template
- Font family ও color theme
- Intro / Outro template
- Thumbnail design style
- Logo / branding

**আলাদা থাকবে (প্রতিটি ভাষায় ভিন্ন):**
- Code implementation → C++ / Java / Python syntax
- Code explanation → Language-specific walkthrough
- Dry run → ভিন্ন উদাহরণ ও data structure
- Intuition → প্রতিটি ভাষার mindset অনুযায়ী rewrite
- Slide text → Copy-paste নয়, সম্পূর্ণ নতুনভাবে লেখা

**Language-specific টিপস:**
- C++ → STL, vectors, pointers, competitive programming angle
- Java → Collections framework, OOP thinking, FAANG tips
- Python → Pythonic approach, built-ins, LeetCode tricks
