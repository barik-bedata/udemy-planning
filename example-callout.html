Color psychology guide for course callouts: when to use blue, yellow, red, and purple

Blue — #2196F3

সাধারণ ব্যাখ্যা, স্টেপ চলছে, নিউট্রাল ইনফো

সাইকোলজি

Blue = বিশ্বাস + শান্ততা। স্টুডেন্টের ব্রেইন রিল্যাক্সড থাকে, নতুন ইনফো সহজে absorb করে। কোনো alert নেই, শুধু "এইটা জানো"।

Yellow — #FFD700

Complexity দেখানো, সতর্কতা, পারফরম্যান্স নোট

সাইকোলজি

Yellow = সতর্কতা + মনোযোগ। "এইখানে একটু থামো" সিগন্যাল। O(N²) বা nested loop দেখলে yellow ব্যবহার করলে স্টুডেন্ট automatically সিরিয়াস হয়।

Red — #FF4D4D

Base case, Error, Edge case, Stack overflow রিস্ক

সাইকোলজি

Red = বিপদ + থামো। ব্রেইনের amygdala সরাসরি activate হয়। শুধু critical condition-এ ব্যবহার করো — overuse করলে effect নষ্ট হয়।

Purple — #A78BFA

Pro-tip, অপ্টিমাইজেশন, ইন্টার্ভিউ ট্রিক, deeper insight

সাইকোলজি

Purple = প্রিমিয়াম + exclusive। "এইটা সবাই জানে না" feeling তৈরি করে। স্টুডেন্ট মনে রাখে কারণ এটা "special" মনে হয়।

def binary_search(arr, target):
  lo, hi = 0, len(arr) - 1
  while lo <= hi:         ← Blue (সাধারণ loop)
    mid = (lo + hi) // 2
    if arr[mid] == target:  ← Yellow (complexity পয়েন্ট)
      return -1             ← Red (base case / not found)
      # use lo+(hi-lo)//2   ← Purple (pro-tip)
INFO

Loop চলছে

lo ≤ hi মানে এখনও খোঁজার জায়গা আছে

O(log N)

Half করছে

প্রতিবার search space দুই ভাগ হয়

BASE CASE

পাওয়া যায়নি

−1 return = not found signal

Quick rule: Blue সবচেয়ে বেশি, Yellow মাঝে মাঝে, Red শুধু critical-এ, Purple সবচেয়ে কম — তাহলে প্রতিটির impact থাকবে।
